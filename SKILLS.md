# SKILLS.md

This is a shadow entry point for the repo skills layer.

The canonical source of truth lives in `skills/README.md`.

## What lives here
- reusable operating procedures
- platform-specific guides for channels, providers, and tools
- structured workflows with verification steps

## Layout
- `skills/<skill>/SKILL.md`
- `skills/<category>/<skill>/SKILL.md`
- `skills/README.md` for shared conventions

If this file and `skills/README.md` ever drift, treat `skills/README.md` as authoritative.
